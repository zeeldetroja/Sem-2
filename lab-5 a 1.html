Concept of WWW

  • It stands for World Wide Web.
  • It is basically system of internet serves that support specially formatted documents.
  • It is a collection of text pages,videos,digital pohtographs and animations we can access over the internet.
  • user access the World Wide Web facilities via client called a web browser.

HTTP Protocol

  1. HTTP for Hyper Transfer Protocol.
  2. HTTP is based on the client-server architecture model and a stateless request/response protocol.
  3. 404 File Not Found is a common HTTP status code.
